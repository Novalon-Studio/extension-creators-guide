# Website listing files

Files in this folder are loaded by [novalon.studio](https://novalon.studio) for the extension library card and detail page.

## Required

- `extension.json` — listing title, summary, overview, install steps, and metadata

## Preview images (recommended)

Add up to three preview images using these exact base names:

- `Image1.png` (or `.jpg`, `.jpeg`, `.webp`)
- `Image2.png`
- `Image3.png`

Missing images show as **No Image Provided** on the website until you add them.
